## Dodger
A fun game where you have to move around using the arrows to escape the enemies that will fall down from the sky

## Overview 
this file will help you create a fully opperating dodging game with the deep understanding of how to code keys and make game templates

## Features
- moving with arrow kays or "WASD"
- lives and scores
- dodging the villans falling from the sky with random shapes, size and speed
- starting background and ending background
- Sound effects(background sound and end of game sound)
- dies when baddie hits the main character

## Structure 
step 2 - making the base for the game, making the introduction background, setting up the boundaries of where and how big the game is going to be played
step 3 - setting of the baddies and the creation of them as well as the main character and codes for the movements it do(ex. codes for the keys needed to be pressed for the main character to move in a certain way)
step 4 - Codes for where the baddie is moving and where it is falling from and the math and mechanism for differentboundaries set in the game, also codes for the scoring and the top score counting
step 5 - overall making finishing touches and fixation of errors and the adding of different sound effects used in the game.

## Requirements : 
- python3
- pygame

## Quick Start
1. Install Requirements : `pip install -r requirements.txt`
2. Run the game : `python main.py`
